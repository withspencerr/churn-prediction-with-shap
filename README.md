# Churn Prediction with SHAP

**Goal:** Predict customer churn and explain model results with SHAP.  
**Tools:** LightGBM · Python · SHAP · scikit-learn · Google Colab  

## Highlights
- Trains a churn prediction model on synthetic data  
- Evaluates AUC and plots ROC curve  
- Visualizes feature importance and impact using SHAP  
- Provides a clear, interpretable story for non-technical audiences  

## Example Output
- AUC ≈ 0.85  
- Top drivers: feature_3, feature_5, feature_9  
- SHAP summary plot shows positive correlation between high values and churn risk  

## How to Reproduce
1. Open `churn_with_shap.ipynb` in **Google Colab**  
2. Click “Run all”  
3. View model results and SHAP visualizations
